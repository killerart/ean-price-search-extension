Setup Instructions:

  1. Get Google API Key:
    • Go to Google Cloud Console
    • Create a new project or select existing one
    • Enable "Custom Search API"
    • Create credentials (API Key)
    • Copy the API key and paste it above
  2. Create Custom Search Engine:
    • Go to Google Custom Search Engine
    • Click "Add" to create a new search engine
    • Enter "*" in "Sites to search" to search the entire web
    • Click "Create" and copy the Search Engine ID
    • Paste the ID in the field above
  3. Test Configuration:
    • Save your settings and click "Test Connection"
    • If successful, the extension is ready to use!

Note: Google Custom Search API provides 100 free searches per day. Additional searches require payment.
